## 安装

`@fta/components` 对等依赖如下：

- `prop-types`
- `classnames`
- `react-motion`
- `Taro`&`React`

在安装`@fta/components`前请确保您的项目中已经安装上述依赖，详见`package.json`

```bash
yarn add prop-types classnames react-motion @fta/components
```
